Initialize connection to core database.

More than one application database can be defined in server conf file. However, only O\+N\+E application database can be active per server mode. If \textquotesingle{}mode\textquotesingle{} attribute is empty, polecat will assume any mode. Otherwise, database is defined for given mode only. The \textquotesingle{}use\textquotesingle{} attribute indicates that the database should be loaded for the respective server mode. Polecat will scan database definitions until it finds one suitable for the current server mode where the \textquotesingle{}use\textquotesingle{} attribute is set. 
\begin{DoxyCode}
<database \textcolor{keywordtype}{id}=\textcolor{stringliteral}{"core"} name=\textcolor{stringliteral}{"polecat"} mode=\textcolor{stringliteral}{"server"} use=\textcolor{stringliteral}{"1"}>
 <dsn>mysql:\textcolor{comment}{//username:password@localhost/databasename</dsn>}
</database>
\end{DoxyCode}


Only one instance of core (server mode) database can be active.
