Return the data model (resource) corresponding to a web request U\+R\+I/path.

Able Polecat expects the part of the U\+R\+I, which follows the host or virtual host name to define a \textquotesingle{}resource\textquotesingle{} on the system. This function returns the data (model) corresponding to request. If no corresponding resource is located on the system, or if an application error is encountered along the way, Able Polecat has a few built-\/in resources to deal with these situations.

N\+O\+T\+E\+: Although a \textquotesingle{}resource\textquotesingle{} may comprise more than one path component (e.\+g. ./books/\mbox{[}I\+S\+B\+N\mbox{]} or ./products/\mbox{[}S\+K\+U\mbox{]} etc), an Able Polecat resource is identified by the first part only (e.\+g. \textquotesingle{}books\textquotesingle{} or \textquotesingle{}products\textquotesingle{}) combined with a U\+U\+I\+D. Additional path parts are passed to the top-\/level resource for further resolution. This is why resource classes validate the U\+R\+I, to ensure it follows expectations for syntax and that request for resource can be fulfilled. In short, the Able Polecat server really only fulfils the first part of the resource request and delegates the rest to the \textquotesingle{}resource\textquotesingle{} itself.
