enlistTransaction (   $transactionClassName,
AblePolecat_Message_RequestInterface  $Message = NULL,
AblePolecat_Registry_Entry_ResourceInterface  $ResourceRegistration = NULL 
)
protected

Load requested transaction and prepare to run as child of transaction in progress.

Able Polecat uses the phrase 'enlist transaction' to mean:

  1. Load child transaction class.
  2. Prepare child transaction to run.
  3. Push on top of current transaction stack.
Parameters
string$transactionClassNameName of transaction class.
AblePolecat_Message_RequestInterface$MessageOptional request to enlisted transaction.
AblePolecat_Registry_Entry_ResourceInterface$ResourceRegistrationOptional, accompanies $Message.
Returns
Instance of AblePolecat_TransactionInterface ready to run.
Exceptions
AblePolecat_Transaction_Exceptionif transaction could not be loaded, prepared or enlisted.

save (   $savepointName)
protected

Create a save point by given name on existing transaction.

Creating a save point will also update the existing transaction. If the transaction has not already been started, calling save in effect starts it.

Parameters
string$savepointNameName of save point.
Returns
string ID of new save point.
