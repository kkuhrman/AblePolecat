static wakeup ( AblePolecat_AccessControl_SubjectInterface  $Subject = NULL)
static

Create a new instance of object or restore cached object to previous state.

Best practice wakeup() will return the object fully initialized and ready to work. Should anything prevent this, wakeup() should throw an exception so as to prevent method calls on non objects or objects that are not properly initialized. Thus, wakeup() should always be called within a try/catch block and chaining is encouraged. For example: try { MyObject::wakeup()->myMethod(); } catch(AblePolecat_Exception $Exception) { AblePolecat_Command_Log::invoke($this, $Exception->getMessage(), AblePolecat_LogInterface::WARNING); }
