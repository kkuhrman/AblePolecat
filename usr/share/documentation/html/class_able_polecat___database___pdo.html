getErrorCount ( )

Helper function returns a count of 'significant' database errors.

NOTE: PDO::errorInfo will return blank error info arrays on some successful DML operations Array(0 => '00000', 1 => null, 2 => null). This function peeks ahead at the internal error store and returns a count of only those errors which are likely to indicate a failed operation or database server internal error.
