Detailed Description

dispatch ( AblePolecat_AccessControl_AgentInterface  $Agent,
AblePolecat_MessageInterface  $Message 
)

Add a message to the queue.

Executive summary of the dispatch process

  1. Get resource registration entry
  2. Check if resource is restricted 2.1 If unrestricted check cache for a timely and relevant response. 2.1.1 If cached response found GOTO 7 2.1.2 If no cached response found GOTO 3 2.2 If restricted check if agent/role has permission to access 2.2.1 If access is permitted GOTO 2.1 2.2.2 If access denied and accessDeniedCode == 403 generate access denied response and GOTO 7 2.2.3 If access denied and accessDeniedCode == 401 generate authentication form response and GOTO 7
  3. Start transaction corresponding to request method/resource
  4. Get response registration entry corresponding to transaction status code and resource object returned by 3
  5. Generate response from data returned by 3 & 4
  6. Update cache with response
  7. Send response.
